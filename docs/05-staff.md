# 5. Panduan Staff

---

## 5.1 Login

1. Buka https://checkin.dsti-ums.id di browser HP atau PC
2. Klik **"Masuk ke Sistem"**
3. Masukkan **Email** dan **Password**
4. Klik **Login**

> **Rekomendasi browser:** Google Chrome atau Firefox terbaru.  
> **Akses dari HP:** Gunakan browser Chrome di Android untuk fitur GPS dan kamera.

---

## 5.2 Dashboard

Setelah login, dashboard menampilkan:
- Status presensi hari ini
- Ringkasan kehadiran bulan ini
- Jadwal kerja aktif

---

## 5.3 Check-in

**Menu:** Sidebar → Presensi / Check-in

### Langkah Check-in
1. Buka halaman **Check-in**
2. **Lokasi GPS** akan dideteksi otomatis
   - Pastikan izin lokasi sudah diberikan di browser
   - Jika GPS gagal, gunakan tombol **Input Manual** untuk memasukkan koordinat
3. **Foto Selfie** (opsional):
   - Klik **Buka Kamera**
   - Izinkan akses kamera
   - Klik tombol kamera untuk mengambil foto
   - Klik **Ambil Ulang** jika ingin foto ulang
4. Isi **Alamat** (opsional) — contoh: `Kantor DSTI`, `WFH dari rumah`
5. Isi **Catatan** (opsional) — contoh: `WFH`, `Meeting luar kantor`
6. Klik **Check-in Sekarang**

### Status Check-in
| Status | Keterangan |
|--------|-----------|
| **Hadir** | Check-in sebelum batas waktu |
| **Terlambat** | Check-in melebihi batas + toleransi |

### Ketentuan
- Check-in hanya bisa dilakukan **sekali per hari**
- Tidak bisa check-in pada **hari libur** yang sudah diatur admin
- Tidak bisa check-in jika ada **cuti yang disetujui** untuk hari tersebut

---

## 5.4 Check-out

**Menu:** Sidebar → Presensi (halaman yang sama dengan Check-in)

Setelah check-in, halaman berubah menjadi form Check-out.

### Langkah Check-out
1. Buka halaman **Presensi**
2. GPS akan dideteksi otomatis
3. Foto selfie (opsional)
4. Isi catatan jika perlu
5. Klik **Check-out Sekarang**

### Status Check-out
| Status | Keterangan |
|--------|-----------|
| **Hadir** | Check-out setelah jam pulang |
| **Pulang Awal** | Check-out sebelum jam pulang yang ditentukan |

---

## 5.5 Riwayat Presensi

**Menu:** Sidebar → Riwayat Presensi

Menampilkan riwayat presensi bulanan dengan filter:
- **Bulan** — pilih bulan yang ingin dilihat
- **Status** — filter berdasarkan status kehadiran

### Kolom Riwayat
| Kolom | Keterangan |
|-------|-----------|
| Tanggal | Tanggal presensi |
| Jam Masuk | Waktu check-in |
| Jam Keluar | Waktu check-out |
| Durasi | Total jam kerja |
| Status | Hadir / Terlambat / Pulang Awal / Izin / Absen |

---

## 5.6 Pengajuan Cuti

**Menu:** Sidebar → Cuti → Ajukan Cuti

### Langkah Pengajuan
1. Klik **+ Ajukan Cuti**
2. Pilih **Jenis Cuti** (Cuti Tahunan, Sakit, Izin, dll.)
3. Pilih **Tanggal Mulai** dan **Tanggal Selesai**
4. Isi **Alasan/Keterangan**
5. Klik **Kirim Pengajuan**

### Status Pengajuan
| Status | Keterangan |
|--------|-----------|
| **Menunggu** | Belum diproses supervisor/admin |
| **Disetujui** | Cuti disetujui, kehadiran otomatis tercatat sebagai Izin |
| **Ditolak** | Cuti ditolak, lihat catatan dari atasan |

### Membatalkan Pengajuan
Pengajuan yang masih berstatus **Menunggu** dapat dibatalkan dari daftar cuti.

---

## 5.7 Riwayat Cuti

**Menu:** Sidebar → Cuti → Riwayat Cuti

Menampilkan semua pengajuan cuti beserta statusnya.

---

## 5.8 Profil

**Menu:** Sidebar → Profil

### Edit Profil
1. Klik **Edit Profil**
2. Ubah **Nama**, **Nomor HP**, atau **Foto Profil**
3. Klik **Simpan**

---

## 5.9 Ganti Password

**Menu:** Sidebar → Ganti Password

1. Masukkan **Password Lama**
2. Masukkan **Password Baru** (minimal 8 karakter)
3. Ulangi **Password Baru**
4. Klik **Simpan**

---

## 5.10 Logout

Klik ikon profil di pojok kanan atas → **Logout**

---

## 5.11 Tips Penggunaan dari HP

- Gunakan **Chrome** atau **Firefox** terbaru
- Akses via **HTTPS** (`https://checkin.dsti-ums.id`) agar GPS berfungsi
- Saat pertama kali check-in, browser akan meminta izin **Lokasi** dan **Kamera** — pilih **Izinkan**
- Jika GPS tidak muncul, pastikan:
  - Lokasi HP diaktifkan (Settings → Location → On)
  - Izin lokasi browser diaktifkan
